# Webový prohlížeč ve WPF s WebView
Takový jednoduchý webový prohlížeč ve WPF s WebView a WPF UI. Nenahradí moderní prohlížeč jako Firefox, Safari apod.
**Obsahuje funkce, jako například**:
- Karty
- Záložky
- Nastavení
- a více!
